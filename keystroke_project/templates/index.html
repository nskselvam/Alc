Keystroke Dynamics for Early Cognitive Decline

This simple web interface demonstrates a "real AI" process in language anyone can understand.

It never records what you type. Instead, it measures how long each key is held and how quickly you move to the next one. These timing signals are fed to a machine-learning model that looks for small changes in typing rhythm—changes that can be an early warning sign of cognitive decline such as mild cognitive impairment or early Alzheimer’s disease.

To see the model at work, click the buttons below. They will request the appropriate endpoint and display the results without leaving this page.
