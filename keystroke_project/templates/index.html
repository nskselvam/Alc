🔑 Keystroke Dynamics Typing Test

AI-Powered Typing Pattern Analysis for Cognitive Health Monitoring
How it works: This test measures your typing rhythm, not what you type. It analyzes timing patterns like how long you hold keys and the time between keystrokes to detect potential cognitive changes. Your privacy is protected - we never record the actual text.

🤖 Machine Learning Algorithm: Random Forest Classifier

What is Random Forest? It's an ensemble learning method that creates multiple decision trees and combines their predictions for more accurate results. Each tree looks at different aspects of your typing pattern.

Features Analyzed:

  • Hold Time: How long each key is pressed (milliseconds)
  • Flight Time: Time between releasing one key and pressing the next
  • Error Rate: Frequency of backspaces and corrections
  • Variability: Consistency in typing rhythm (standard deviation)

Model Specs: 100 decision trees, trained on 500+ sessions with both healthy and cognitive-decline patterns
